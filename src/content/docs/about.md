这是一个个人技术博客，用来整理嵌入式、AIoT 和 AI Agent 项目的学习笔记。

当前重点项目是 Pixel Soul：一个基于 ESP32-S3-RLCD-4.2 的桌面 AIoT 设备，包含显示、音频、语音识别、云端 Agent 和本地交互逻辑。

## 为什么写

很多调试结论如果只停留在聊天记录或临时日志里，很快就会丢失。这个博客希望把它们整理成更稳定的文章：

- 能解释“为什么这样设计”。
- 能复现“如何一步步验证”。
- 能沉淀“下次遇到类似问题怎么处理”。
